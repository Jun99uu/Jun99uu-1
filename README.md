# 🐸Caculator🐸
![image](https://user-images.githubusercontent.com/44965706/159973478-71b966c1-aa22-4c4f-bcf6-8a0047aa8e10.png)
### 자바스크립트를 이용하여 작성된 기본적인 Caculator입니다. <br/>
숫자를 문자열로 입력받아, 계산을 진행할 때는 Number를 통해 자료형을 변환해줍니다. <br/>
